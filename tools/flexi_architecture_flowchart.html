Architecture Components Description

  • User Interaction Layer - Web browsers and mobile devices where users interact with the application
  • Network Layer - NGINX reverse proxy handling routing and SSL termination
  • Frontend Layer - React Native/Expo application with multiple UI routes/screens
  • Backend Layer - Node.js/Express API server with controllers and services
  • Database Layer - Two PostgreSQL databases managed through Prisma ORM

Key Features

  • Security: HTTPS traffic through Nginx
  • Scalability: Containerized services that can be scaled independently
  • Modularity: Clear separation between frontend, backend, and database layers
  • Multiple Databases: Specialized databases for different domains (core app vs ads)
  • Modern Frontend: React Native/Expo app with provider-based state management
  • API-First Design: Well-structured API clients for different domains
  • Internationalization: Support for multiple languages (English and Thai)
